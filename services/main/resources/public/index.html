boot-gae Modules Sample App

This is the static index.html page for the default module (called "main") of the boot-test sample app.

Test the main.core servlet of the default module:

Modules:

  • greetings (port: 8088)

Warning: servlet startup is excruciatingly slow on the dev server, but once loaded, page refresh is pretty fast.

Editing

You'll need to run the `gae/watch` task in each module subproject.

Then edit a source file and refresh the webpage. You should see the changes almost instantly.
