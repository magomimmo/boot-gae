boot-gae Services Sample App

This is the static index.html page for the default service (called "main") of the boot-gae Services sample app.

Test the main.core servlet of the default service:

  • main

Services:

  • greetings (port: 8088)
  • upload (port: 8089)

Warning: servlet startup is excruciatingly slow on the dev server, but once loaded, page refresh is pretty fast.

Editing

You'll need to run the `gae/watch` task in each service subproject.

Then edit a source file and refresh the webpage. You should see the changes almost instantly.
